# 자가면역질환 전조증상 자가진단 시스템

자가면역질환의 전조증상을 자가진단할 수 있는 웹 애플리케이션입니다.

## 기능

### 1. 공통 전조증상 자가진단 (0~5점)
- 피로감 점수
- 불안감, 우울감, 집중력 점수
- 입맛 저하, 소화불량 점수
- 관절통 점수
- 피부 이상(붉은기, 건조, 가려움) 점수

### 2. 질환별 전조 증상 자가진단
- **류마티스 관절염**: 통증 부위 체크
- **건선**: 피부 부위별 증상 체크
- **크론병**: 배변 횟수, 형태 체크, 복통 위치 체크
- **제1형 당뇨병**: 혈당체크, 인슐린 투여 시간 및 용량 체크
- **다발성 경화증**: 시야 흐림/감각 둔화 여부 체크, 보행 거리 및 시간 체크
- **루푸스**: 얼굴 발진/입안 궤양 여부 체크, 햇빛 노출 시간 체크 → 기상데이터(자외선 수치)
- **쇼그렌 증후군**: 눈물/침 분비량 체크
- **자가면역성 갑상선 질환**: 맥박, 체온, 체중 변화 체크, 불면/괴민/무기력 정도 체크

## 설치 및 실행

### 필수 요구사항
- Node.js 18 이상
- npm 또는 yarn

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 빌드
```bash
npm run build
```

## 기술 스택
- React 18
- TypeScript
- Vite
- Axios (기상 API용)

## 환경 변수 설정

### 기상청 API 키 설정 (환경 정보 기반 Flare 위험도 기능)
1. [공공데이터포털(data.go.kr)](https://www.data.go.kr)에서 회원가입
2. "기상청_단기예보 ((구)_동네예보) 조회서비스" 또는 "기상청_초단기실황조회" API 신청
3. `.env` 파일 생성 및 API 키 추가:
   ```
   VITE_KMA_API_KEY=your_api_key_here
   ```
4. 현재 설정된 지역: **대전광역시 유성구**

### 루푸스 자외선 수치 API (선택사항)
- OpenWeatherMap API 키를 사용하려면:
  ```
  VITE_WEATHER_API_KEY=your_api_key_here
  ```

## 주의사항
- 이 자가진단은 참고용이며, 정확한 진단을 위해서는 전문의와 상담하시기 바랍니다.
- 기상청 API 키가 없으면 예시 데이터를 사용합니다.
